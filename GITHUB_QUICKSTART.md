# GitHub Pages Quick Start (10 Minutes)

## Why GitHub Pages?
- Professional URL: `https://yourusername.github.io/eudoria-reference`
- Easy to update (just upload new files)
- Version control built-in
- Free forever

## Step-by-Step:

### 1. Create GitHub Account (2 minutes)
- Go to https://github.com
- Click "Sign up"
- Choose a username (this will be in your URL)
- Verify your email

### 2. Create New Repository (1 minute)
- Click the "+" icon (top right) → "New repository"
- Repository name: `eudoria-reference`
- Description: "Interactive reference for the Eudoria universe"
- Make it **Public** ✓
- Don't initialize with README (we already have one)
- Click "Create repository"

### 3. Upload Your Files (3 minutes)
- On the new repository page, click "uploading an existing file"
- Drag and drop ALL these files and folders:
  - `index.html`
  - `script.js`
  - `styles.css`
  - `README.md`
  - `HOSTING_GUIDE.md`
  - `main.js`
  - `package.json`
  - `images/` folder (with all god portraits)
  - `Eudoric Zodiac/` folder (with all zodiac images)
- Commit message: "Initial upload - Eudoria Regional Reference"
- Click "Commit changes"

### 4. Enable GitHub Pages (2 minutes)
- Click "Settings" tab (top of repository)
- Click "Pages" in the left sidebar
- Under "Source", select "main" branch
- Click "Save"
- Wait 1-2 minutes for deployment

### 5. Get Your Link! (1 minute)
- GitHub will show your URL at the top of the Pages settings
- It will be: `https://[your-username].github.io/eudoria-reference/`
- Click the link to test it
- **Share this link with your beta readers!**

## To Update Later:
1. Go to your repository
2. Click on the file you want to edit (e.g., `script.js`)
3. Click the pencil icon (top right)
4. Make your changes
5. Scroll down and click "Commit changes"
6. Wait 1-2 minutes - your site updates automatically!
